<object>

<object> contains a description of a single identifiable physical object. [13.3.6. Objects]
Module namesdates
Contained by
namesdates: listObject
May contain
core: note
linking: linkGrp
namesdates: objectName
Note
Our use of object is not TEI conformant, but the break is fairly trivial: where objectName is simply used instead of objectIdentifier/objectName.
